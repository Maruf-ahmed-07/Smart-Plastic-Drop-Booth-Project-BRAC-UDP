# Smart Plastic Drop Booth Project - Cox's Bazar

A multi-page project website for the Smart Plastic Drop Booth initiative by Team EcoChange Makers in collaboration with **BRAC UDP**.

## About This Project

This project focuses on improving plastic waste collection in Cox's Bazar through a practical and scalable booth-based system. Instead of replacing the existing waste chain, it strengthens the current BRAC UDP model by introducing visible plastic-only drop points in schools, roadside public areas, and other high-footfall locations.

The idea is built around behavior change, easier public participation, and a more sustainable collection flow for low-value plastic that is often ignored. The website presents the full project story, including the problem context, solution logic, implementation process, expected outcomes, financial model, gallery, and team information.

## Website Structure

- **Home:** Project overview and impact
- **Problem:** Context and environmental challenges in Cox's Bazar
- **Solution:** Smart Plastic Drop Booth concept and design logic
- **Implementation:** Rollout process, field use, and collection flow
- **Outcomes:** Expected environmental, social, and system-level results
- **Finance:** Detailed 24-month financial model
- **Team:** Team members, acknowledgements, and supporters
- **Booth Gallery:** Example booth placement visuals
- **FAQ:** Common questions and answers
