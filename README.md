# EcoChange Makers Project Site

React + Vite site for the Smart Plastic Drop Booth Project.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploying to Vercel

1. Push this repository to GitHub.
2. Import the GitHub repo in Vercel.
3. Leave the build command as `npm run build`.
4. Leave the output directory as `dist`.
