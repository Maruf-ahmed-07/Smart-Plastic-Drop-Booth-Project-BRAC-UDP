# Smart Plastic Drop Booth Project — Cox's Bazar

A comprehensive landing website for the Smart Plastic Drop Booth initiative by Team EcoChange Makers in collaboration with **BRAC UDP**.

## 📌 About This Project

This is an innovative waste management solution designed to improve plastic waste collection in Cox's Bazar. Instead of creating a new system, we strengthen the existing BRAC UDP infrastructure by introducing **Smart Plastic Drop Booths** at strategic locations (schools, roadsides, beachside areas).

**Key Features:**
- Plastic-only collection design
- Behavior-based sustainable approach  
- Financially self-sustaining through sponsor model
- Scalable from 4 → 10 → 30 booths over 24 months
- Integrated with existing BRAC waste collection chain

## 🎯 Why Smart Plastic Drop Booths?

Traditional awareness campaigns create temporary change. Our solution focuses on **behavior change and habit formation** by making proper plastic disposal:
- ✅ Easy and convenient
- ✅ Visible and memorable
- ✅ Integrated with existing systems
- ✅ Financially sustainable
